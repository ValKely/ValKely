Credit Card Approval Prediction

This project addresses the challenge of predicting credit card approvals using machine learning to support financial institutions in managing risk and improving customer satisfaction.

What it Solves

Automated Credit Decisions

Reduces manual effort and errors in the approval process using predictive models.

Risk Management

Helps financial institutions manage risk by identifying potential defaulters early.

Improved Customer Experience

Enables quicker and fairer credit card decisions using data-driven insights.

My Role & Process

Data Preprocessing

Cleaned and encoded data by handling missing values and converting categorical features.

Model Building

Used Logistic Regression, Decision Tree, KNN, Random Forest, and XGBoost models.

Voting Classifier

Combined predictions of multiple classifiers to improve overall performance.

Handling Class Imbalance

Applied SMOTE to balance classes and ensure fairness in prediction.

Key Features

Feature Engineering

Converted categorical data, handled missing values, and created a cleaned dataset.

Model Evaluation

Measured accuracy, used cross-validation, and compared classifiers' performance.

Best Performing Model

The Voting Classifier achieved 91% accuracy and demonstrated robustness.

Tools I Used

Python

For data preprocessing, model training, and evaluation.

Pandas & Scikit-learn

Used for data manipulation, model building, and cross-validation.

SMOTE

Handled class imbalance using oversampling of the minority class.
