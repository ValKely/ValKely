Jumia Web Scraping Analysis

Discover the Real Drivers Behind Discount Pricing on Smartphone Products

What it Solves

Discount Strategies Unveiled

Jumia's discount decisions often appear random. This project identifies the key factors influencing price changes.

Key Influencers

Brand reputation, product condition, ratings, and product type are the driving forces behind discount rates.

Improving Sales Insights

By understanding these patterns, sellers can optimize their strategies to maximize sales potential.

My Role & Process

Data Scraping

Used Python to scrape smartphone listings from Jumia, gathering key attributes such as brand, ratings, and prices.

Data Cleaning

Normalized inconsistent data, removed duplicates, and extracted meaningful discount information.

Visualization

Created Power BI dashboards to reveal insights and correlations between factors such as product type and discount rate.

Key Features

Insightful Data Analysis

Pinpoints exact factors influencing Jumia's discount strategies for smartphones.

Visual Data Representation

Charts and graphs provide clear, digestible insights into discount trends.

Easy to Understand

Simple, clear language makes data-driven findings accessible to non-experts.

Tools I Used

Python

Used for web scraping, data cleaning, and analysis.

Pandas & NumPy

Libraries for efficient data manipulation and numerical analysis.

SQL

For data cleaning, transformation, and filtering.

Power BI

For building dynamic dashboards and visualizing key trends and insights.
